# Credits

## AI-generated game art

The current game uses project-specific AI-generated raster assets created for this app:

- MathHero player sprite
- Slime King, Magma Ogre, and Cyber Dragon boss sprites
- Math magic projectile and impact asset
- Neon math academy arena background

The generated image assets are stored in `public/assets/generated/`.

## Code-generated support art

Small boss projectile fallback textures and UI effects are generated locally in Phaser.
