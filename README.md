# Math Blaster: Boss Academy

## What is this?

2D mobile browser educational math boss fight game for primary school children.

## Tech stack

- Phaser
- TypeScript
- Vite
- Vitest
- Vercel
- AI-generated game art stored locally in `public/assets/generated`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm run test
```

## Deploy on Vercel

Build command: `npm run build`

Output directory: `dist`
